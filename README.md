# Canyon Crown Tree - Website

Brand new website for **Canyon Crown Tree**, a professional tree care and removal company based in Salt Lake City, Utah. Customized from the Guerra Plastic Surgery template with top tree care industry patterns for design, UX, and lead generation. Optimized for SEO.

- **Services:** Tree Removal, Pruning & Trimming, Stump Grinding, Emergency Services, Tree Health Care, Tree Planting

## Website Features

- **Modern, responsive design** with nature-inspired color palette (greens, browns, earth tones)
- **Mobile-first UX** with sticky navigation, hamburger menu, and floating CTA
- **12 pages** including homepage, about, 6 individual service pages, gallery, blog, and contact
- **Lead generation focused** with prominent CTAs, free quote forms, and phone numbers on every page
- **SEO optimized** with Schema.org LocalBusiness markup, meta tags, sitemap, and keyword-rich content
- **Trust signals** including ISA certification, insurance info, testimonials, and project gallery
- **Educational blog** with tree care tips for Utah homeowners
- **24/7 emergency service** prominently featured

## Tech Stack

- Semantic HTML5
- CSS3 with custom properties and responsive design
- Vanilla JavaScript (no frameworks)
- Schema.org structured data (JSON-LD)
- Google Fonts (Inter)

## Design Patterns Applied

Based on analysis of top tree care websites:
- Clean, uncluttered layouts with nature-inspired colors
- High-quality imagery of work, teams, and equipment
- Responsive mobile-first design
- Prominent contact info and CTAs on every page
- Detailed service pages with galleries
- Educational blog content for SEO and engagement
- Trust signals (certifications, insurance, reviews)
- Local SEO focus with Schema markup and location keywords
